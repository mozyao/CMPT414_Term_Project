####################################
##### How to execute the codes #####
####################################
1. open a new notebook in Google Colab
2. mount google drive
3. upload "CMPT414_Project.zip" onto "/content/drive/My Drive" 
4. execute the following code in the colab notebook to unzip

	 $%cd /content/drive/My Drive
	 $!unzip CMPT414_Project.zip

5. execute code in "setup.ipynb" in the "CMPT414_Project" folder to download and setup the dataset
6. upload the model onto "CMPT414_Project/model"
	the link of the model: https://drive.google.com/file/d/114CwHX-Rapvv_xw3EqrdpAiSBGKjhkTT/view?usp=sharing 
7. execute code in 'Train_Data.ipynb'

	Training:
		execute code from "Training 1" to "Training 5" in 'Train_Data.ipynb', the process may take roughly 1 hour

	Demo:
		execute code "Traning 1", "Traning 3", and "Demo 1" to "Demo 3" in 'Train_Data.ipynb'
